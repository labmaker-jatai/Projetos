# Régua do LabMaker IFG Jataí

Nesta pasta estão disponíveis os arquivos necessários para a fabricação da régua do Laboratório Maker do IFG Jataí.

A régua possui 24 cm de comprimento, cantos semiarredondados e foi projetada para corte a laser em MDF de 3 mm de espessura.

Entre os arquivos disponíveis, encontram-se:

* o arquivo em formato `.svg`; e
* o arquivo `.wcvr`, compatível com o software AutoLaser, facilitando o envio para as cortadoras a laser Delta CNC.
