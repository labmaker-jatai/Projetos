# Jogo das Frações

Arquivos para a fabricação do clássico jogo de frações, utilizado no ensino desse conteúdo. Esta versão inclui as frações em braile (código Nemeth), tornando o material acessível a alunos com deficiência visual.

Os setores circulares foram projetados para corte a laser em chapas de MDF de 3 mm de espessura.

## Arquivos disponíveis

- **.svg** – arquivo vetorial para uso geral
- **.wcvr** – arquivo compatível com o software AutoLaser, pronto para envio às cortadoras a laser Delta CNC
